# Kubernetes Inside Out

> Master Kubernetes from core concepts to real-world operations with hands-on practice.

---

## Overview

**Kubernetes Inside Out** is a hands-on guide designed for engineers who want to understand Kubernetes deeply. It focuses on practical cluster operations, troubleshooting, and DevOps practices, bridging the gap between theory and production-ready skills.

---

## Features

* Kubernetes architecture
* Core workloads (pods, deployments, services)
* Networking & storage concepts
* Security basics (RBAC, secrets)
* Troubleshooting scenarios
* Real-world DevOps practices
* EKS

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sam-osung/Kubernetes-Inside-Out
cd kubernetes-inside-out
```

Start exploring from the `01-Architecture/` folder.
